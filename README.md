# tcga-stad-network-analysis
# Molecular and Structural Reprogramming of Gastric Cancer Revealed by Systems-Level Transcriptomic Analysis
This repository contains processed results and analysis code for: "Molecular and Structural Reprogramming of Gastric Cancer Revealed by Systems-Level Transcriptomic Analysis".

## Data sources
- TCGA-STAD RNA-seq and clinical data were obtained from the Genomic Data Commons (GDC).
  Raw data are available from: https://gdc.cancer.gov/

## Repository contents
- `data_processed/` Processed tables generated in this study
- `results/` Network/enrichment outputs
- `scripts/` Analysis scripts/notebooks used to generate results
- `figures/` Generated figures (if applicable)

## How to reproduce (high level)
1) Download TCGA-STAD data from GDC (see manuscript Methods).
2) Run scripts in `scripts/` in the order described there.
3) Outputs will appear in `results/` and `figures/`.
